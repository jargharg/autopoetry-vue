# autopoetry-vue

## Project setup

Create a .env file and add Guardian Dev API key as VUE_APP_API_KEY, e.g.

```
VUE_APP_API_KEY=xxx_xx_xxxx_
```

Install dependencies with

```
yarn
```

### Compile and hot-reload for development

```
yarn serve
```

### Compile and minify for production

```
yarn build
```
